Class: GameWatcher

GameWatcher

The wrapper in the station, exchanges message with gamepads.

new GameWatcher(url)

To construct a GameWatcher object on the station, which can handle messages from Gamepad, the GameWatcher won't connect to the WebSocket by default, please call GameWatcher.connect() before use.
Parameters:
Name Type Description
url The server url to connect.

sendMessage(msgs, uuid, datas)

Send message to the gamepad,
Message contain three parts and is defind as:
{"msg": message, "data": {"uid" : uuid, "msg": msgs, "data" : datas}}
Only use this when you want to implement more functions because the message is sent automatically.
