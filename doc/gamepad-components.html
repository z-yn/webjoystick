slider

Sliders is a handle on a rail, the handle can be dragged on the rail, and on each drag event, it will send message to the station with type "sliderMove" or "sliderStart", messages are defined as:
{"id": slider.id, "position": x}
And when the drag is end, it will send message with type "sliderEnd" and without position
The position is normalized in the range(-1, 1).
. The id is the DOM element's id.
Sliders should have class name "slider" and tag name "div", and call UIManager.init() or UIManager.initSlider() then the handle and events will be added
The handle should have class name "slider-handle".

stick

Stick is a circular handle in a larger basing circle, the handle can be dragged to any position in the circle, and on each drag event, it will send message to the station with type "dragMove" or "dragStart", messages are defined as:
{"id": stick.id, "position":{"x": x, "y": y}}
The x, y in the position is normalized in the range(-1, 1).
The id is the DOM element's id.
And when the drag is end, it will send message with type "dragEnd" and without position.
Sticks' basing circle should have class name "stick" and tag name "div", and call UIManager.init() or UIManager.initSticks() then the handle and events will be added
The auto added handle would have class name "stick-handle".
